##  Sobre o Projeto

O **Impacta.dev** nasceu de um paradoxo do mercado de tecnologia: juniores precisam de experiência para conseguir o primeiro emprego, mas poucas empresas contratam sem experiência.

Do outro lado, milhares de ONGs realizam trabalhos incríveis, mas são invisíveis digitalmente por falta de recursos para contratar profissionais de tecnologia.

**Nós somos a ponte.** O Impacta.dev cria squads voluntários onde desenvolvedores, designers e gestores de projeto iniciantes ganham experiência real, mentoria e um portfólio de impacto, enquanto ONGs recebem sites e sistemas profissionais gratuitamente.

###  Funcionalidades Principais

* **Páginas Segmentadas:** Jornadas claras e distintas para "Desenvolvedores" e "ONGs", com dores e benefícios específicos para cada público.
* **Formulários de Triagem (Formspree):** Captação de leads e inscrições integrada diretamente ao e-mail, sem necessidade de backend complexo.
* **Design System:** Uma identidade visual elegante, quente e humana, que foge do padrão frio de sites de tecnologia.
* **Acessibilidade (VLibras):** Widget de tradução para LIBRAS integrado, garantindo um site inclusivo.
* **PWA (Progressive Web App):** Configurável para ser instalado como aplicativo em dispositivos móveis.
* **Animações Fluidas (GSAP):** Micro-interações e efeitos de scroll (parallax, fade-in) que tornam a navegação mais agradável.

##  Tecnologias Utilizadas

O projeto foi construído com foco na solidez dos fundamentos da web, priorizando performance, semântica e organização.

* **HTML5 Semântico:** Estrutura clara e acessível, utilizando tags apropriadas para cada conteúdo.
* **CSS3 Moderno:**
    * **Variáveis CSS (`:root`):** Para um Design System consistente e fácil manutenção de cores e fontes.
    * **Flexbox & Grid:** Para layouts responsivos e complexos.
    * **Media Queries:** Adaptação fluida para todos os tamanhos de tela (Mobile First).
* **JavaScript (ES6+):**
    * **Manipulação do DOM:** Para menus interativos, FAQs e lógica de formulários.
    * **GSAP (GreenSock Animation Platform):** Para animações de alta performance e controle de scroll (ScrollTrigger).
* **Ferramentas & Integrações:**
    * **Formspree:** Para gerenciamento e envio de formulários de contato.
    * **VLibras:** Widget de acessibilidade.
    * **SVG:** Ícones vetoriais leves e escaláveis.

##  Decisões de UX/UI

Como Tech Lead e Designer do projeto, cada elemento foi pensado para transmitir confiança e propósito:

1.  **Tom de Voz Humano:** A copy foge de termos técnicos frios e foca no impacto humano ("Códar com propósito", "Tecnologia Social").
2.  **Paleta de Cores Quente:** O uso de tons de creme, laranja e marrom cria uma sensação de acolhimento e comunidade, diferente do azul/preto padrão de tech.
3.  **Arquitetura de Informação:**
    * **Triagem na Home:** A página inicial serve como um hub que direciona rapidamente o usuário para o seu caminho ideal (Dev ou ONG).
    * **Páginas Específicas:** Cada público tem uma página dedicada que fala a sua língua e resolve suas dúvidas específicas.
4.  **Transparência Radical:** Uma seção de FAQ robusta em cada página para eliminar objeções antes do cadastro.

##  Como Rodar Localmente

Este é um projeto estático, o que torna a execução muito simples.

1.  Clone este repositório:
    ```bash
    git clone [https://github.com/anajuliatoriani/impacta-dev.git](https://github.com/anajuliatoriani/impacta-dev.git)
    ```
2.  Navegue até a pasta do projeto:
    ```bash
    cd impacta-dev
    ```
3.  Abra o arquivo `index.html` no seu navegador de preferência.

> **Dica:** Para a melhor experiência de desenvolvimento e para evitar problemas com carregamento de recursos (como o GSAP ou VLibras em alguns casos), recomenda-se usar uma extensão como o **Live Server** no VS Code.
